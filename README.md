# Présentation des differentes methodes de génération procédural.

|NOM|DESCRIPTION|
|-|-|
|Simple Room Placement|Place aléatoirement des 'rooms' sur une carte données, s'assure qu'aucune room ne se superpose et les lient entre eux par un 'chemin'|
|BSP| BSP pour Binary Space Partition découpe une carte en deux et place dans chaque côté une room, se répete le nombre de fois voulus.
|Cellular Automata| Fonctionne comme le 'white noise' placement totalement aléatoire de point selon une densité preset|
